## Overview

### What is it

NetMapper is an online tool for the CLI tool `nmap`. 
You can choose which of these three options you want to use to scan the network:
    - `-sV`
    - `-sS`

You can also choose between options like:
    - Host Timeout
    - Max Rate
    - Max Retries
